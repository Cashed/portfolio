Star Chat

Star Chat is a web app for those who find that sometimes the only way to truly express themselves is with an appropriate Star Trek picture. This app transports the user into a total Star Trek communication experience.

    Tech Stack
  • HTML/CSS
  • jQuery
  • Firebase API

Runnit

Runnit is a Reddit clone app built on AngularJS. I had been playing a lot of Shadowrun on the SNES while learning Angular and I fell in love with how much work went into building the world within the game. This is my homage.

    Tech Stack
  • Bootstrap
  • CSS3
  • AngularJS
  • SoundCloud API

MEAN Pots

This is a potion shopping app built on the MEAN stack. Users are presented with inventory that they may filter based on category and/or keyword. Potions can be added to the 'bag of holding' (cart) and then checked out.

    Tech Stack
  • Bootstrap
  • AngularJS
  • NodeJS
  • ExpressJS
  • MongoDB

FlyBeats

FlyBeats is a 3D web game where the user must navigate a phoenix around blocks that pulsate to music. This is a full-stack web application built as a capstone project during my time at Galvanize.

    Tech Stack
  • Three.js
  • Web Audio API
  • AngularJS
  • NodeJS
  • ExpressJS
  • PostgreSQL

Infinite Garden

Infinite Garden is a collaborative project which seeks to give visitors a refuge away from online noise, and to observe the organic processes encoded in the environmental flora. Procedural algorithms, such as Perlin Noise, are used to generate the scene.

    Tech Stack
  • Unity (WebGL)
  • Turn.js
  • AngularJS
  • NodeJS
  • ExpressJS
  • PostgreSQL
